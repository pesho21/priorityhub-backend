<h1 align="center"> PriorityHub API </h1>

## Стартиране на API-то

Ако искате да стартирате само и единствено api-то, използвайте тези команди:

```
# Clone repo
git clone https://github.com/pesho21/priorityhub-backend.git

# Project setup
npm install

# Development
npm run start

# Watch mode
npm run start:dev

# Production mode
$ npm run start:prod
```
